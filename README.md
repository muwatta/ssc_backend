# SSC Cooperative System Backend

A Django-based backend system for managing cooperative societies, financial contributions, loans, investments, and member operations.

## Features
- JWT Authentication (SimpleJWT)
- Staff ID based login system
- Member management and approval workflow
- Savings and loan tracking
- Suretyship system
- Investment module
- Notification system
- Supabase PostgreSQL integration

## Tech Stack
- Django 5
- Django REST Framework
- PostgreSQL (Supabase)
- SimpleJWT
